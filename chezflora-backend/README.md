# ChezFlora Backend API

API backend pour l'application ChezFlora, une plateforme de vente de fleurs et plantes en ligne.

## Technologies

- Node.js
- TypeScript
- Express.js
- Sequelize ORM
- MySQL
- Redis
- Docker

## Installation

### Prérequis

- Node.js (v14+)
- MySQL
- Redis
- Docker et Docker Compose (optionnel)
